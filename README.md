# Personal Portfolio Website

## Overview

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website showcases personal information, skills, projects, and contact details in a clean and modern design.

The portfolio is designed to provide visitors with an overview of my background, technical skills, and project experience while demonstrating front-end web development abilities.

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* Smooth scrolling navigation
* Modern user interface
* About Me section
* Skills section
* Projects showcase
* Contact form
* Hover effects and animations
* Scroll reveal animations using JavaScript
* Clean and organized layout

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Sections Included

### Home

Introduces the portfolio owner with a profile image, name, and short description.

### About Me

Provides a brief overview of personal background, interests, and goals.

### Skills

Displays technical skills and tools used in web development.

### Projects

Showcases completed projects with descriptions and highlights.

### Contact

Allows visitors to send messages through a contact form.

## How to Run the Project

1. Download or clone the project files.
2. Open the project folder.
3. Open `index.html` in any modern web browser.

Alternatively, use the Live Server extension in Visual Studio Code for a better development experience.
